# HTML Advanced

This project focuses on learning HTML5 structure and semantic tags.

## Learning Objectives
- How to create the skeleton of an HTML5 page
- How to use semantic HTML tags to structure a web page
- How to use headings and follow hierarchical order
- How to make lists in HTML
- How to structure data in a table
- How to integrate video and audio files
- How to correctly structure an HTML page

## Project: Techium
Techium is the company name used throughout these webpages.

## Requirements
- Code should be W3C compliant
- No CSS styling in this project
